# QUICK START GUIDE FOR REVIEWERS
# License to Live: MIAS

## FOR PROFESSOR/REVIEWERS - 5 MINUTE SETUP

This guide gets you running the system in 5 minutes.

---

## STEP 1: INSTALL R PACKAGES (2 minutes)

Open RStudio and run:

```r
install.packages(c("shiny", "DBI", "RMySQL", "shinyjs", 
                   "stringr", "lubridate", "DT", "ggplot2", 
                   "gridExtra", "dplyr", "scales"))
```

Wait for installation to complete.

---

## STEP 2: DOWNLOAD PROJECT FILES (1 minute)

Download all files from the shared location (OneDrive/GitHub) to a folder.

Example: C:/MIAS_Review/

---

## STEP 3: SET WORKING DIRECTORY (30 seconds)

In RStudio console:

```r
setwd("C:/MIAS_Review")  # Change to your folder path
```

---

## STEP 4: LAUNCH THE SYSTEM (30 seconds)

```r
source("launch_mias.R")
```

You'll see a menu with 4 options.

---

## TROUBLESHOOTING

**Problem:** Package installation fails
**Solution:** Install packages one at a time

**Problem:** "Cannot find file 'launch_mias.R'"
**Solution:** Check working directory with getwd()

**Problem:** Database connection fails
**Solution:** Check internet connection, credentials are correct

**Problem:** App launches but shows "No data available"
**Solution:** This is normal - use medical_info_manager.R to add test data
